Fuse brought interface design and native application development into one quick feedback loop. Apps were described in declarative `.UX` markup, scripted with JavaScript, and extended through Uno and native platform code.

The same component model covered layout, animation, interaction, navigation, and reusable controls. A live preview made the source feel closer to a visual design tool while still producing native iOS and Android applications.

## Across the toolchain

I worked on Fuse from 2015 to 2018. The work moved between desktop tooling, framework code, documentation, native integrations, and the many languages underneath it: C#, C++, Objective-C, Java, JavaScript, TypeScript, Uno, and `.UX`.

I also demonstrated and taught Fuse internationally, helping designers and developers understand a model intended to give both groups meaningful access to the finished application.
