<p class="about-lede">I’m Andreas Kennedy Rønning. I’ve spent more than twenty years making software, tools, and interactive work.</p>

I started in graphics and animation, became a developer by accident, and carried that practice through games, exhibitions, creative tools, enterprise systems, and public-sector software.

I care about the craft: how tools are designed, how people use them, and how good work gets made. I remain a proud forever amateur.

## A few things I’ve done

- Led development of a 40LOC React user interface on secure Norwegian healthcare-planning data managed in ServiceNow.
- Built safety and location-tracking applications for field personnel.
- Created VSCode tooling for building React frontends on ServiceNow SP and UI16.
- Helped build Fuse’s cross-platform UX toolchain (iOS, Android, Win and OSX).
- Built visual-authoring tools for orchestrating expo installations.

## Speaking, tools, and open source

I've been an outspoken FOSS supporter, created [**DConsole**](https://github.com/furusystems/DConsole), contributed and evangelzed for the Haxe ecosystem, spoke to developers and creatives at **Flash on the Beach** and **Reasons to be Creative**, and organized international Fuse workshops. An [early, three-minute elevator pitch](https://vimeo.com/15740065) survives.

Older work can be found on [GitHub](https://github.com/sunjammer). I occasionally comment on [LinkedIn](https://www.linkedin.com/in/andreas-kennedy-r%C3%B8nning-ab823867/); My music is on [Bandcamp](https://sunjammer.bandcamp.com/) and [SoundCloud](https://soundcloud.com/doomsday).
