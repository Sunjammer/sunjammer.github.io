Super Croissant Fighter is a browser game built around the ancient conflict between pastry and pastry.

The web build runs directly in the browser. Click inside the game canvas to enable audio.

## Play it

Use the **Open project** link above to launch the game. A downloadable Windows build is also linked from the game page.
